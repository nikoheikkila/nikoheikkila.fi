# Reflect and Learn from Current or Past Context

When this command runs, follow the steps below:

1. Recall the project memory and task context.
2. Study the changes made in this session.
3. Make note of any friction, obstacles, and workarounds you encountered while working on this task.
4. Write all the repository-wide instructions that would have been helpful to you into @AGENTS.md
5. Write all the amendments to specific agentic skills that would have been helpful to you into @.claude/skills/
6. Add any concrete developer experience improving assets into the repository including new scripts, configuration, and
   data.
7. Present a summary of your lessons learned to the user.
